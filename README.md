# NLP project

Este proyecto consta de 3 partes (3 npm packages) que corren independientemente:
1. Crypto Analyzer
1. Results Generator
1. Results Compiler

![diagram](./diagram.png?raw=true "System Diagram")

## Setup (en cada package)
1. instalar node (I'm using v14.17.0)
1. correr `npm install`
1. correr `npm start`
